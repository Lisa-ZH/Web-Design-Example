Images / Image Hover Effect

Hover Image Animation

Zoom In (Bigger)

.zoom-in img{
transition: all .6s ease-in-out;
}

.zoom-in:hover img{
transform: scale(1.25);
}

Zoom Out (Smaller)

.zoom-out img{
transform: scale(1.25);
transition: all .6s ease-in-out;
}

.zoom-out:hover img{
transform: scale(1);
}

Pan Up

.pan-up img{
transform-origin: top;
transform: scale(1.2) translate(0,0);
transition: all .6s ease-in-out;
}

.pan-up:hover img{
transform: scale(1.2) translate(0,-15%);
}
