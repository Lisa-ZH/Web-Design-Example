Escape HTML Characters

To show code on an HTML page as text (not executed), you need to escape HTML characters and usually wrap the code in <pre> and <codee> tags.

Example

        
    <meta http-equiv="refresh" content="0; url=https://www.houstonmethodistcareers.org/culture-and-engagement.html">
